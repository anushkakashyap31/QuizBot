# QuizBot

# Problem Statement	- Non-Profit Quiz/Tutor Bot
# Description -	Develop an interactive AI-driven educational bot for the Non-Profit domain. The application engages users in targeted assessments based on 'Donor Emails', evaluating their responses using a
 Large Language Model. Beyond simple scoring, the bot provides deep contextual explanations for incorrect answers, creating a personalized learning loop that bridges knowledge gaps and reinforces industry-specific concepts.
# Major Tools -	Python, Vector DB, LLM API
